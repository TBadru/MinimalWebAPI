 minimal API with ASP.NET Core with a Swagger UI.

## Setup

1. **Clone the repository**
    ```bash
    git clone https://github.com/TBadru/MinimalWebAPI.git
    cd MinimalWebAPI
    ```

3. **Run the application/start the server**
    ```bash
    dotnet clean
    dotnet build
    dotnet run
    ```
    The API testing page generated by swagger  will be available at http://localhost:<port>/swagger.
